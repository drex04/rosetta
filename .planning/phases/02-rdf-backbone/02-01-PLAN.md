  <action>
1. Install dependencies:
   ```
   npm install n3 vite-plugin-node-polyfills
   npm install --save-dev @types/n3
   ```

2. Add polyfills to `vite.config.ts`:
   ```ts
   import { nodePolyfills } from 'vite-plugin-node-polyfills'
   // add to plugins array: nodePolyfills()
   ```

3. Create `src/types/index.ts` with:
   ```ts
   export interface PropertyData {
     uri: string
     label: string
     range: string       // e.g. "xsd:float" or full URI for ObjectProperty
     kind: 'datatype' | 'object'
   }

   export interface ClassData {
     uri: string
     label: string
     prefix: string
     comment?: string
     properties: PropertyData[]
   }

   export type OntologyNode = import('@xyflow/react').Node<ClassData, 'classNode'>

   export interface SubclassEdgeData {
     predicate: 'rdfs:subClassOf'
   }

   export interface ObjectPropertyEdgeData {
     uri: string
     label: string
     predicate: 'owl:ObjectProperty'
   }

   export type OntologyEdge =
     | import('@xyflow/react').Edge<SubclassEdgeData, 'subclassEdge'>
     | import('@xyflow/react').Edge<ObjectPropertyEdgeData, 'objectPropertyEdge'>
   ```

4. Create `src/lib/rdf.ts`:
   - `parseTurtle(text: string): Promise<{ nodes: OntologyNode[], edges: OntologyEdge[] }>`
     - Use `N3.Parser` to parse into `N3.Store`
     - For each `?s rdf:type owl:Class` triple: create a node with `id = "node_" + localName(s.value)`
     - For each `?p rdf:type owl:DatatypeProperty` with `rdfs:domain ?s`: add to that class's `properties[]` as `kind: 'datatype'`
     - For each `?p rdf:type owl:ObjectProperty` with `rdfs:domain ?s, rdfs:range ?o` where both s and o are classes: create an `objectPropertyEdge`
     - For each `?s rdfs:subClassOf ?o`: create a `subclassEdge`
     - Node positions default to `{ x: 0, y: 0 }` (overlaid by caller)
   - `canvasToTurtle(nodes: OntologyNode[], edges: OntologyEdge[]): Promise<string>`
     - Use `N3.Writer` with turtle format
     - Emit `@prefix` declarations derived from unique prefixes in node URIs
     - For each node: emit `?s a owl:Class` + `rdfs:label` + properties
     - For each subclassEdge: emit `?s rdfs:subClassOf ?o`
     - For each objectPropertyEdge: emit `?p a owl:ObjectProperty; rdfs:domain ?s; rdfs:range ?o`
   - Helper `localName(uri: string): string` — extracts local name after `#` or last `/`

5. Write `src/__tests__/rdf.test.ts`:
   - `parseTurtle round-trip`: parse a small Turtle snippet → call canvasToTurtle → re-parse → verify same number of nodes/edges
   - `parseTurtle error`: passing invalid Turtle should reject/throw
   - `localName`: unit tests for URI → local name extraction
  </action>
  <verify>
    - `npm run test -- rdf` passes all cases
    - TypeScript compiles without errors: `npx tsc --noEmit`
  </verify>
  <done>
    `parseTurtle` converts valid Turtle to OntologyNode[] + OntologyEdge[]. Round-trip test passes. Invalid Turtle throws catchable error.
  </done>
