   - Export `ViolationRecord` interface: `{ id: string; sourceId: string; targetClassUri: string; targetPropUri: string | null; message: string; severity: string; canvasNodeId: string | null }`.
     NOTE: `sourceId` is populated in `index.ts` (validateSource has access to `source.id`); set to `''` as placeholder in `validateWithShacl`.
   - Export `validateWithShacl(dataStore: N3.Store, shapesStore: N3.Store): Promise<ViolationRecord[]>` (async — rdf-validate-shacl's validate() returns a Promise):

Set `canvasNodeId: null` and `sourceId: ''` (both resolved later). Assign `id: crypto.randomUUID()`.
     - Wrap entire call in try/catch; rethrow with a descriptive message so callers can surface an error banner.

<task type="tdd">
  <name>Task 2: shapesGenerator.ts + instanceGenerator.ts</name>
  <files>src/lib/shacl/shapesGenerator.ts, src/lib/shacl/instanceGenerator.ts, src/__tests__/shacl.test.ts</files>
  <action>
**shapesGenerator.ts:**
Write `generateShapes(ontologyNodes: OntologyNode[]): N3.Store`:
- Constants needed: `SH = 'http://www.w3.org/ns/shacl#'`, `RDF_TYPE = 'http://www.w3.org/1999/02/22-rdf-syntax-ns#type'`, `XSD = 'http://www.w3.org/2001/XMLSchema#'`.
- For each `OntologyNode` with `data.uri`:
  - Mint shape URI = `data.uri + 'Shape'`.
  - Add triple: `shapeUri rdf:type sh:NodeShape`.
  - Add triple: `shapeUri sh:targetClass classUri`.
  - For each `prop` in `data.properties`:
    - If `prop.kind === 'datatype'`:
      - Expand short range: strip `'xsd:'` prefix and append to `XSD` (e.g. `'xsd:float'` → `XSD + 'float'`). If range doesn't start with `'xsd:'`, skip (e.g. ObjectProperty ranges treated as `sh:class`).
      - Mint blank node for the property constraint.
      - Add: `blankNode sh:path propUri`.
      - Add: `blankNode sh:datatype datatypeUri`.
      - Add: `shapeUri sh:property blankNode`.
    - If `prop.kind === 'object'`:
      - Mint blank node; add `sh:path` + `sh:class prop.range` (range is already a full class URI for ObjectProperty).
      - Add: `shapeUri sh:property blankNode`.
- Return the populated store.

**instanceGenerator.ts:**
Write `jsonToInstances(json: string, schemaNodes: SourceNode[]): N3.Store`:
- Constants: `RDF_TYPE`, `XSD`.
- Derive `uriBase` from `schemaNodes[0]?.data.prefix ?? ''`.
- `JSON.parse(json)` — wrap in try/catch, return empty N3.Store on failure.
- Walk the parsed value with a recursive `walkValue(value, className, depth)` function (depth limit 10):
  - If `value` is an array: call `walkValue(item, className, depth)` for each item.
  - If `value` is a plain object: mint a blank node `instance`. Add `instance rdf:type <uriBase + className>`. For each field `[key, val]`:
    - If `val` is a primitive: add `instance <uriBase + key> typedLiteral(val)` where `typedLiteral` uses `N3.DataFactory.literal` with XSD datatype matching `xsdRangeShort`.
    - If `val` is a plain object: recurse `walkValue(val, toPascalCase(key), depth+1)` → get child blank node; add `instance <uriBase + key> childBN` (ObjectProperty).
    - If `val` is an array: recurse `walkValue(val, toPascalCase(key), depth+1)`.
    - Skip null/undefined.
  - Return the blank node.
- `xsdRangeShort` and `toPascalCase` are local helpers (same logic as in jsonToSchema — do NOT import from jsonToSchema since it is not exported).

  <name>Task 3: constructExecutor.ts + shacl/index.ts</name>
  <files>src/lib/shacl/constructExecutor.ts, src/lib/shacl/index.ts, src/__tests__/shacl.test.ts</files>
  <action>
**constructExecutor.ts:**
Write `executeConstruct(instanceStore: N3.Store, mappings: Mapping[]): N3.Store`:
- Constants: `RDF_TYPE`.
- Create a result `N3.Store`.
- For each mapping:
  - Find all subjects matching `? rdf:type <mapping.sourceClassUri>`: `instanceStore.getSubjects(RDF_TYPE, mapping.sourceClassUri, null)`.
  - For each subject, find all objects for `<subject> <mapping.sourcePropUri> ?`: `instanceStore.getObjects(subject, mapping.sourcePropUri, null)`.
  - For each (subject, val) pair:
    - Mint a blank node `target = N3.DataFactory.blankNode(subject.value + '_target')`.
    - Add `target rdf:type <mapping.targetClassUri>`.
    - Add `target <mapping.targetPropUri> val`.
- Return result store.

**shacl/index.ts:**
Export `ViolationRecord` (re-export from validator.ts).
Write `validateSource(source: Source, ontologyNodes: OntologyNode[], mappings: Mapping[]): Promise<ViolationRecord[]>` (async):
0. Early-return `[]` if `source.schemaNodes.length === 0` (no schema → no URI base → results would be meaningless false-positives).
1. `const shapesStore = generateShapes(ontologyNodes)`.
2. `const instanceStore = jsonToInstances(source.json, source.schemaNodes)`.
3. `const dataStore = executeConstruct(instanceStore, mappings)`.
4. `const violations = await validateWithShacl(dataStore, shapesStore)` — let throw propagate.
5. For each violation, resolve `canvasNodeId` and set `sourceId`:
   - Set `v.sourceId = source.id`.
   - Find mapping where `m.targetPropUri === v.targetPropUri && m.targetClassUri === v.targetClassUri`.
   - Find canvas node where `n.data.uri === mapping?.sourceClassUri`.
   - Set `v.canvasNodeId = node?.id ?? null`.
6. Return violations.
