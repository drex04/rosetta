<task type="tdd">
  <name>Task 1: validationStore.ts</name>
  <files>src/store/validationStore.ts</files>
  <action>
Create `src/store/validationStore.ts` using Zustand (no persist — ephemeral by design):

```typescript
interface ValidationState {
  // per-source results keyed by sourceId
  results: Record<string, ViolationRecord[]>
  loading: boolean
  stale: boolean
  error: string | null
  lastRun: number | null

  runValidation: () => Promise<void>
  setStale: (stale: boolean) => void
  reset: () => void
}
```

`runValidation`:
0. If `get().loading` return early (guard against double-click / concurrent calls).
1. `set({ loading: true, error: null })`.
2. Read `useSourcesStore.getState().sources`, `useOntologyStore.getState().nodes`, `useMappingStore.getState().getMappingsForSource`.
3. For each source, call `await validateSource(source, ontologyNodes, getMappingsForSource(source.id))` wrapped in try/catch.
   - On catch: `set({ loading: false, error: e.message ?? 'Validation failed', stale: false })` and return early.
4. Build `results: Record<string, ViolationRecord[]>` from all sources.
5. `set({ results, loading: false, stale: false, lastRun: Date.now() })`.

`setStale`: `set({ stale: true })`.
`reset`: `set({ results: {}, loading: false, stale: false, error: null, lastRun: null })`.

**Stale subscription** — export a `subscribeValidationToMappings()` function that calls `useMappingStore.subscribe(() => useValidationStore.getState().setStale(true))` and returns the unsubscribe handle. This is called once in App.tsx on mount.

**Tests** (add to `src/__tests__/shacl.test.ts` or new `src/__tests__/validationStore.test.ts`):
- After calling `setStale(true)`, `stale === true`.
- After calling `reset()`, `results` is `{}` and `stale` is `false`.
- `subscribeValidationToMappings` sets stale=true when mappingStore changes (mock `useMappingStore.subscribe` or trigger `addMapping`).
  </action>
