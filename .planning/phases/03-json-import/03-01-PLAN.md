<task type="auto">
  <name>Task 2: SourceSelector pill bar</name>
  <files>src/components/layout/SourceSelector.tsx</files>
  <action>
Replace the placeholder with a functional pill bar:
1. Read `sources`, `activeSourceId`, `addSource`, `removeSource`, `setActiveSourceId`, `updateSource` from sourcesStore.
2. Render a horizontal flex row of pills — one per source, active pill highlighted (amber ring).
3. Each pill: click body → setActiveSourceId. Click `×` → if source.json is non-empty, open a shadcn AlertDialog ("Delete source and all its schema nodes?" with Cancel/Delete buttons) before calling removeSource. If json is empty, delete immediately.
4. Pill name: double-click → enter inline edit mode (controlled input). Enter/blur commits via updateSource; if committed value is empty or whitespace-only, revert to the previous name (do NOT call updateSource). Esc cancels and reverts.
   IMPORTANT — Esc+blur guard: use an `isEscaping` ref. On Esc keydown: set `isEscaping.current = true`, revert value, call input.blur(). In onBlur: if `isEscaping.current` is true, reset flag and return without committing.
5. `+` button at end: auto-name = find smallest N ≥ 1 where `'Source N'` is not already in sources.map(s=>s.name). Calls addSource with `{ id: generateSourceId(), name: 'Source N', order: sources.length, json: '', schemaNodes: [], schemaEdges: [] }`, sets it active, and immediately puts the new pill into inline-edit mode.
6. Use Tailwind + shadcn tokens for styling. Active pill: amber-500 background. Inactive: muted.
  </action>
  <verify>
- Adding a source creates a new pill in inline-edit mode
- Renaming commits on Enter/blur, cancels on Esc
- Deleting a non-active source removes it; deleting active source switches to first remaining
- `npm run build` passes
  </verify>
  <done>SourceSelector functional with add/rename/delete inline. Matches must_haves truth: pill bar with + and ×.</done>
</task>
