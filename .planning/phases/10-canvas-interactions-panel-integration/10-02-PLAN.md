In `useCanvasData.ts`:

1. **Split into two memos** (performance: selection changes must not rebuild all edge objects):

   ```ts
   // Base memo — structural data only. Rebuilds on mapping/node/source changes.
   const mappingEdgesBase = useMemo((): MappingFlowEdge[] => {
     // ... existing loop building edges with mappingId, groupId, groupOrder, kind
     // Do NOT include selectedMappingId here
   }, [mappings, masterNodes, sources])

   // Selection memo — annotates selected:bool. Rebuilds only on selectedMappingId change.
   const selectedMappingId = useMappingStore((s) => s.selectedMappingId)
   const mappingEdges = useMemo(
     () => mappingEdgesBase.map(e => ({
       ...e,
       selected: (e.data as { mappingId?: string })?.mappingId === selectedMappingId,
       data: {
         ...(e.data as object),
         selected: (e.data as { mappingId?: string })?.mappingId === selectedMappingId,
       },
     })),
     [mappingEdgesBase, selectedMappingId]
   )
   ```

   data: {
     mappingId: mapping.id,
     groupId: mapping.groupId,
     groupOrder: mapping.groupOrder,
     kind: mapping.kind,   // NEW — for badge rendering in MappingEdge
   },
   ```

Note: React Flow uses the top-level `selected` prop for selection styling. The selection memo sets it on the outer edge object.

   ```

3. Add `onEdgeClick` handler (REQ-103):
   ```ts
   const handleEdgeClick = useCallback(
     (_: React.MouseEvent, edge: Edge) => {
       if (edge.type === 'mappingEdge') {
         const mappingId = (edge.data as { mappingId?: string } | undefined)?.mappingId
         if (mappingId) {
           setSelectedMappingId(mappingId)
           setActiveRightTab('MAP')
         }
       }
     },
     [setSelectedMappingId, setActiveRightTab]
   )
   ```

4. Add `onNodeDoubleClick` handler (REQ-104, REQ-105). This must also call the existing `onStartEdit` logic from 10-01. The handler fires after 10-01's `data.onStartEdit(nodeId)` is injected into node data, so we call both:
   ```ts
   const handleNodeDoubleClick = useCallback(
     (_: React.MouseEvent, node: Node) => {
       // Tab navigation (10-02)
       if (node.type === 'classNode') {
         setActiveRightTab('ONTOLOGY')
       } else if (node.type === 'sourceNode') {
         setActiveRightTab('SOURCE')
       }
       // Inline edit (10-01) — onStartEdit is injected into node.data by the
       // existing nodeData memo; calling it here keeps all canvas-level effects
       // in one place. Guard in case 10-01 hasn't been built yet.
       const onStartEdit = (node.data as { onStartEdit?: (id: string) => void } | undefined)?.onStartEdit
       if (onStartEdit) onStartEdit(node.id)
     },
     [setActiveRightTab]
   )
   ```

5. Pass both handlers to the `<ReactFlow>` component:
   ```tsx
   <ReactFlow
     ...
     onEdgeClick={handleEdgeClick}
     onNodeDoubleClick={handleNodeDoubleClick}
     ...
   />
   ```

**Important:** If `OntologyCanvas` already has an `onNodeDoubleClick` prop or handler from 10-01, merge — do not create a duplicate. The tab-switch logic belongs here; 10-01's `onStartEdit` call also belongs here per the locked CONTEXT decision.
