Replace the browser-native `window.prompt` rename flow with a fully inline edit experience on ClassNode and SourceNode. Double-clicking a node header reveals stacked label+URI inputs; double-clicking a property row reveals label+dataType inputs. The context menu "Rename" triggers the same inline mode. Add a ReactFlow Panel with visible "+ Ontology Class" / "+ Source Class" buttons for node creation discoverability. All edits sync bidirectionally to the Turtle editor via the existing `onCanvasChange` path.
</objective>

/** Update a node's label, URI, or properties array. Fires onInvalidateMappings

**Define `onCommitEdit` for ontology nodes:**
```ts
const handleCommitOntologyEdit = useCallback((
  nodeId: string,
  patch: { label?: string; uri?: string } | { propertyUri: string; propPatch: Partial<PropertyData> }
) => {
  // Capture pre-edit snapshot for revert on serialization failure
  const preEditNodes = useOntologyStore.getState().nodes
  if ('propertyUri' in patch) {
    updateProperty(nodeId, patch.propertyUri, patch.propPatch)
  } else {
    updateNode(nodeId, patch)
  }
  // Trigger canvasToTurtle sync; revert store on failure
  const { nodes: latestNodes, edges: latestEdges } = useOntologyStore.getState()
  try {
    onCanvasChange?.(latestNodes, latestEdges)
  } catch {
    setNodes(preEditNodes)  // revert canvas to pre-edit state
    toast.error('Edit failed — Turtle serialization error, changes reverted')
  }
}, [updateNode, updateProperty, onCanvasChange])
```

**Define `onCommitEdit` for source nodes:**
```ts
const handleCommitSourceEdit = useCallback((nodeId: string, patch: { label?: string; uri?: string }) => {
  // Find which source owns this node
  const { sources } = useSourcesStore.getState()
  const ownerSource = sources.find(s => s.schemaNodes.some(n => n.id === nodeId))
  if (!ownerSource) return
  updateSchemaNode(ownerSource.id, nodeId, patch)
  const updatedSource = useSourcesStore.getState().sources.find(s => s.id === ownerSource.id)
  if (updatedSource && onSourceCanvasChange) {
    onSourceCanvasChange(updatedSource.schemaNodes, updatedSource.schemaEdges ?? [])
  }
}, [updateSchemaNode, onSourceCanvasChange])
```

**Inject into node data** (in the existing `useMemo`/`useEffect` that injects `onContextMenu`):
