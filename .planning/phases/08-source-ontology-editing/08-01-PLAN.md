<task type="tdd">
  <name>Task 2: xmlToSchema converter</name>
  <files>src/lib/xmlToSchema.ts, src/__tests__/xmlToSchema.test.ts</files>
  <action>
  1. Create `src/lib/xmlToSchema.ts` exporting `xmlToSchema(xmlString: string, sourceName: string): SchemaResult`. Import `SchemaResult` from `jsonToSchema.ts` (export the type if not already exported).
  2. Use `new DOMParser().parseFromString(xmlString, 'application/xml')`. Check for `parsererror` element — if found, return `{ nodes: [], edges: [], turtle: '', warnings: ['Invalid XML: ...'] }`.
  3. Walk the DOM tree recursively:
     - Element with child elements → `rdfs:Class` (PascalCase from `localName`)
     - Leaf element with text → `DatatypeProperty` on parent class, infer XSD type from text content (reuse `inferXsdType` pattern from jsonToSchema or extract shared util)
     - XML attribute → `DatatypeProperty` with `@` prefix in label
     - Nesting → `ObjectProperty` edge between parent and child classes
     - Repeated siblings with same tag → use first occurrence only (array detection)
  4. Generate nodes (type `sourceNode`, amber scheme), edges, and Turtle string using same N3.Writer pattern as jsonToSchema. Use source-specific URI prefix: `http://example.org/{sourceName}#`.
  5. Write tests: (a) simple flat XML → correct classes + properties, (b) nested XML → ObjectProperty edges, (c) XML with attributes → `@attr` properties, (d) invalid XML → warnings, (e) repeated elements → single class.
  </action>
  <verify>`npm run test` — all xmlToSchema tests pass. Output SchemaResult has same shape as jsonToSchema output.</verify>
  <done>XML source data produces an RDFS schema with classes, datatype properties, and object property edges identical in structure to JSON-generated schemas.</done>
</task>

<task type="tdd">
  <name>Task 3: Format detection utility</name>
  <files>src/lib/detectFormat.ts, src/__tests__/detectFormat.test.ts</files>
  <action>
  1. Create `src/lib/detectFormat.ts` exporting:
     - `detectFormatFromContent(text: string): 'json' | 'xml' | 'unknown'` — trim whitespace, check first char: `{` or `[` → json, `<` → xml, else unknown.
     - `detectFormatFromFile(file: File): 'json' | 'xml' | 'unknown'` — check `file.type` first (application/json, text/xml, application/xml), fall back to extension (.json, .xml), fall back to 'unknown'.
  2. Write tests covering: JSON string, XML string, empty string, file with MIME type, file with extension only, ambiguous content.
  </action>
  <verify>`npm run test` — detectFormat tests pass.</verify>
  <done>Pasting or uploading content correctly detects JSON vs XML format.</done>
</task>
